# 프론트엔드 UI 개편 및 리팩토링 결과 (2026-05-10)

투박했던 기존 UI를 현대적이고 풍성한 디자인으로 전면 교체하였으며, 프로젝트 구조를 직관적으로 개선했습니다.

## 주요 변경 사항

### 1. 폴더 구조 개선
- `src/static` → `src/frontend`로 폴더명 변경 (더 직관적인 명칭 사용)
- `src/api.py` 내의 모든 경로 참조(`static` → `frontend`) 업데이트 및 마운트 경로 수정

### 2. UI/UX 디자인 업그레이드
- **Dark Mode First**: 눈이 편안한 프리미엄 다크 테마 기본 적용 (라이트 모드 토글 지원).
- **Glassmorphism**: 배경 애니메이션(Ambient Blobs)과 반투명 패널을 사용한 깊이 있는 디자인.
- **Rich Sidebar**: 
    - 소크라테스식 질문 깊이 조절 (Light, Standard, Deep)
    - 세션 통계 (대화 횟수, 참조 문서 수)
    - 현재 학습 계획 (마크다운 지원)
    - 참조한 자료 리스트업
- **Enhanced Chat Experience**:
    - **Markdown Rendering**: AI 답변 내의 코드 블록, 리스트 등이 깔끔하게 표시됨.
    - **Welcome Banner**: 빠른 시작을 위한 질문 칩 제공.
    - **Animations**: 부드러운 메시지 등장 효과 및 로딩 인디케이터.
